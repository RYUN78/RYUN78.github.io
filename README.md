# 🍢 みどり亭 モバイルオーダーシステム（MOS）

居酒屋向けのモバイルオーダーシステムです。  
バックエンド不要・サーバーレスで動作するフロントエンドプロトタイプです。

---

## 📱 デモ

`index.html` をブラウザで開くだけで動作します。インストール不要。

---

## 🗂 ファイル構成

```
.
├── index.html           # エントリーポイント
├── style.css            # 全画面のスタイル（レスポンシブ対応）
├── data.js              # メニューデータ（品名・価格・カテゴリ）
├── state.js             # 共有状態・カート操作・画面遷移
├── events.js            # 画面ごとのイベント振り分け
├── clock.js             # 日本時間時計・ラストオーダー・飲み放題タイマー
├── screen_checkin.js    # 入店画面（人数入力）
├── screen_menu.js       # メニュー画面・カテゴリ切り替え
├── screen_qty.js        # 数量選択・カート追加・注文確定トースト
├── screen_cart.js       # カート一覧
├── screen_history.js    # 注文履歴
├── screen_call.js       # 呼び出し・会計確認
└── screen_error.js      # エラー画面
```

---

## ✨ 機能一覧

### 注文フロー
- **入店画面** — 大人・子供の人数を入力して入店（大人0名・子供のみはエラー）
- **メニュー画面** — カテゴリサイドバーで切り替え、商品をタップして数量選択
- **カート** — 数量変更・削除・注文確定
- **注文履歴** — 確定済み注文の一覧と合計金額

### スタッフ呼び出し・会計
- **店員呼び出し** — 呼び出し中画面を表示
- **会計** — 卓番とお支払金額を表示

### 時計・タイマー（全画面右上に常時表示）
- **日本時間（JST）** — 秒単位でリアルタイム表示
- **ラストオーダータイマー** — 残り時間をカウントダウン（30分以内でオレンジ点滅）
- **飲み放題タイマー** — 飲み放題プランを注文確定すると残り時間を表示（10分以内で警告）

## ⚙️ カスタマイズ

### ラストオーダー時刻を変える
`clock.js` の先頭を編集してください。

```js
const LAST_ORDER_TIME = '22:00'; // ← ここを変更
```

### メニューを追加・変更する
`data.js` の `MENU_DATA` を編集してください。

```js
"カテゴリ名": [
  { id: 99, name: "商品名", price: 500, img: "🍺", desc: "説明文" },
],
```

### 飲み放題プランを追加する
`data.js` の `"飲み放題"` カテゴリに `nomihodaiMinutes` を設定してください。

```js
"飲み放題": [
  { id: 37, name: "飲み放題（90分）",  price: 1500, img: "🍻", nomihodaiMinutes: 90  },
  { id: 38, name: "飲み放題（120分）", price: 2000, img: "🍻", nomihodaiMinutes: 120 },
],
```
